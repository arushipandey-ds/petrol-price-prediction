# Petrol Price Prediction

This is a beginner Machine Learning project created using Python and Streamlit.
The project predicts petrol price based on city code, fuel code and quantity.

## About the Project
This project is built for learning purposes while studying Data Science.
A Linear Regression model is used to predict petrol price.

## Dataset
The dataset includes:
- City_Code
- Fuel_Code
- Quantity
- Price

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## Model Used
- Linear Regression (Supervised Learning)

## How It Works
1. User enters city code, fuel code and quantity
2. Model predicts petrol price
3. Result is displayed on screen
